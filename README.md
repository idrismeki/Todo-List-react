# Todo-List-react

Une application Todo List simple et moderne développée avec React.

## Description

Ce projet permet de gérer facilement une liste de tâches.  
L’utilisateur peut ajouter, supprimer et marquer des tâches selon les fonctionnalités intégrées dans l’application.

## Fonctionnalités

- Ajouter une tâche.
- Supprimer une tâche.
- Marquer une tâche comme terminée.
- Interface simple et responsive.
- Gestion locale des données si `localStorage` est utilisé.

## Technologies utilisées

- React
- JavaScript
- HTML5
- CSS3

## Utilisation

- Saisis une tâche dans le champ prévu.
- Clique sur le bouton d’ajout.
- Gère ensuite tes tâches avec les actions disponibles.

## Améliorations possibles

- Ajouter l’édition des tâches.
- Ajouter des filtres : tout, en cours, terminées.
- Sauvegarder les tâches dans le navigateur.
- Ajouter le mode sombre.
- Améliorer l’interface utilisateur.
